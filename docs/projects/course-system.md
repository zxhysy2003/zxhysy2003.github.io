# 智能课程学习系统项目复盘

## 项目背景

本项目围绕智能课程学习系统展开，主要目标是为用户提供课程学习、学习记录、个性化推荐和学习分析等功能。

## 技术栈

- 后端：Spring Boot
- 数据库：MySQL
- 缓存：Redis
- 知识图谱：Neo4j
- 推荐算法：Python + Surprise SVD

## 核心功能

- 课程学习
- 学习行为记录
- 个性化推荐
- 知识图谱展示
- 学习分析

## 实现难点

- 课程、用户、学习记录之间的数据关系需要清晰建模，避免后续统计和推荐逻辑变复杂。
- 推荐系统涉及 Java 后端与 Python 算法侧的协作，需要设计稳定的数据输入和结果输出方式。
- 知识图谱展示依赖课程知识点之间的关系质量，前期建模会直接影响最终展示效果。

## 复盘收获

通过这个项目，我对后端分层设计、推荐系统、缓存设计和知识图谱建模有了更完整的理解。

后续如果继续完善，可以重点补充接口设计、数据库表结构、推荐流程和知识图谱建模过程，让复盘更适合沉淀为完整项目文档。
